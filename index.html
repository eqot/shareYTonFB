shareYTonFB

A bookmarklet to share an YouTube content on Facebook.

Download this project as a .zip file Download this project as a tar.gz file

What is this?

This is a bookmarklet to share an YouTube content with a start time on Facebook.

This bookmarklet is based on the Facebook's official bookmarklet and customized to add a start time so that your friends can watch a particular scene of the YouTube content.

How to use?

  1. Drag and drop this bookmarklet to the toolbar in the browser
  2. Go to the YouTube site and watch any content
  3. Press the bookmarklet in the toolbar
